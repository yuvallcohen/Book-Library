# 📚 Book Library – מערכת לניהול ספרייה

פרויקט זה הוא מערכת לניהול ספרייה המאפשרת צפייה, ניהול ורכישה של ספרים.  
המערכת בנויה בארכיטקטורת **Client–Server** וכוללת צד לקוח וצד שרת עם חיבור למסד נתונים.

---

## 🎯 מטרת הפרויקט
לאפשר ניהול ספרייה דיגיטלית בצורה נוחה:
- הצגת ספרים
- הוספה ומחיקה של ספרים
- סינון וחיפוש ספרים
- חיבור למסד נתונים לצורך שמירת המידע

## ⚙️ מבנה כללי של הפרויקט
- Client – אפליקציית React להצגת הספרייה וניהול הספרים
- Server – שרת Node.js המטפל בלוגיקה ובתקשורת עם MongoDB
- Database – MongoDB לשמירת נתוני הספרים
