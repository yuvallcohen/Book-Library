# 📚 Book Library – מערכת לניהול ספרייה

פרויקט זה הוא מערכת לניהול ספרייה דיגיטלית המאפשרת צפייה, ניהול ורכישה של ספרים.  
המערכת בנויה בארכיטקטורת **Client–Server** וכוללת צד לקוח וצד שרת עם חיבור למסד נתונים.

---

## 🎯 מטרת הפרויקט

לאפשר ניהול נוח ויעיל של ספרייה דיגיטלית, הכולל:
- הצגת ספרים  
- ניהול סל קניות  
- סינון וחיפוש ספרים  
- שמירה וניהול מידע באמצעות מסד נתונים

## ✨ פיצ'רים עיקריים

- ➕ הוספת ספרים לסל הקניות  
- 🗑️ הסרת ספרים מסל הקניות  
- 📂 ניהול נתוני ספרים באמצעות מסד הנתונים  
- 🔍 סינון וחיפוש ספרים בספרייה  
- 🔗 חיבור מלא בין צד הלקוח, צד השרת ומסד הנתונים  

---

## ⚙️ מבנה כללי של הפרויקט

- Client – אפליקציית React להצגת הספרייה וניהול סל הקניות  
- Server – שרת Node.js המטפל בלוגיקה ובתקשורת עם MongoDB באמצעות Mongoose  
- Database – מסד נתונים MongoDB לשמירת נתוני הספרים
